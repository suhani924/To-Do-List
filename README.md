# Real-Time Todo Management System

## Project Overview

Real-Time Todo Management System is a simple task management web application developed using Node.js, Express.js, EJS, HTML, CSS, and Bootstrap.

This application allows users to create, update, delete, and manage tasks efficiently through a user-friendly dashboard.

---

## Features

* Add New Tasks
* Edit Existing Tasks
* Delete Tasks
* Change Task Status
* View All Tasks
* Dashboard with Task Statistics
* Responsive User Interface

---

## Technologies Used

* Node.js
* Express.js
* EJS
* HTML
* CSS
* Bootstrap 5

## Installation Steps

### Step 1

Install Node.js on your system.

### Step 2

Open terminal in project folder.

### Step 3

Install required packages.

npm install

### Step 4

Run the application.

node app.js

### Step 5

Open browser and visit:

http://localhost:3000

---

## Functionalities

### Add Task

Users can create a new task by entering:

* Task Title
* Description
* Priority

### Edit Task

Users can update task details.

### Delete Task

Users can remove tasks permanently.

### Status Management

Task status can be changed from:

Pending → In Progress → Completed

### Dashboard

Displays:

* Total Tasks
* Pending Tasks
* Completed Tasks

---

## Conclusion

The Real-Time Todo Management System is a beginner-friendly CRUD application that demonstrates the use of Express.js routing, EJS templates, and basic task management functionality.
